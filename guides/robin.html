ROBIN

로빈

평철 + 와셔 + 원피스 커넥트 기반의 침대. Q/K (1630) 사이즈 기준 가이드. 5박스 구성으로 출고됩니다.

📋 7 단계 ⏱ 소요 40~60분 👥 인원 2인 🔧 공구 드라이버 + 비트
📄 원본 PDF
⚠️ 조립 3대 원칙
  • 모든 철물을 처음부터 꽉 조이지 말 것. 가볍게 모두 채운 후, 마지막에 수평을 맞추며 꽉 조인다.
  • 힘을 너무 가하면 부서질 수 있으니 천천히 고정한다는 생각으로 돌릴 것.
  • 조립 후 흔들림 체크 필수. 흔들리면 현장에서 점검·수평 재조정.

설치 단계

  1. 파운데이션 상·하부 — 다리 발통 설치

    파운데이션 상판 상·하부를 모두 뒤집어, 유도 보링에 맞추어 다리 발통을 피스로 설치합니다.

    ⛔ Q/K 중앙 다리 주의

    Q/K 파운데이션의 경우 중앙 다리 보링이 되어 있지 않습니다. 중앙에 직접 박아야 하므로 별도 치수 안내(1-1 페이지)를 참고하세요.

  2. 상부 + 하부 결합

    다리 발통이 결합된 파운데이션 상부와 하부를 평철 → 와셔 → 볼트 순서대로 사용해 단단히 조여 고정합니다.

    평철 ─ 와셔 ─ 볼트

  3. 헤드 + 파운데이션 결합

    헤드와 파운데이션을 결합하기 위해, 헤드 뒷면에서 원피스 커넥트를 박아 확실하게 고정합니다.

    원피스 SS 8ea 원피스 Q/K 8ea

  4. 패널 선반 — 다리 발통

    패널 선반을 뒤집은 후, 유도 보링에 맞추어 다리 발통을 피스로 고정합니다.

  5. 패널 + 선반 결합

    패널과 선반을 결합하기 위해 패널 뒷면에서 원피스 커넥트를 박아 견고히 고정합니다.

  6. 헤드 + 패널 연결

    헤드와 패널을 표기된 지점에서 평철 → 와셔 → 볼트 순으로 안정적으로 연결합니다.

    평철 ─ 와셔 ─ 볼트

  7. 최종 조임 + 흔들림 체크

    처음에 가볍게 채워둔 모든 볼트를 수평을 맞추며 차례대로 꽉 조여 마무리합니다.

    ✓ 인수 전 체크
    • 양 끝에서 흔들어 보아 좌우 흔들림 없는지
    • 다리 4개 모두 바닥에 닿아 수평인지
    • 헤드–패널 결합부 평철 정상
