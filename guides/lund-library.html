룬드 라이브러리

2~3단의 다양한 형태 책장·교구장. 라픽스 + 미니픽스 + 목다보 조합 기반.

📄 원본 PDF (6종)
⛔ 임팩트 드라이버 사용 금지

미니픽스·라픽스 결합 시 임팩트 드라이버의 과도한 토크로 부품 파손이 다수 발생합니다. 반드시 수동 드라이버로 결합.

6가지 형태

3단 반원 책장
측판 A·B · 중간선반 2
3단 교구장
우라잡이 6 · 스크류 3
3단 책장
뿔다보·평다보 각 2
3단 코너 책장
측판 좌·우 1개씩 + 앞판
2단 반원 책장
중간선반 1
2단 교구장
우라잡이 4 · 스크류 2

공통 조립 순서

  1. 하판 + 측판 결합 (미니픽스 + 목다보)

    하판과 측판을 미니픽스 4 set + 목다보 4 ea로 결합합니다.

  2. 중간 칸막이 + 뒷판

    중간 칸막이를 결합한 후 뒷판을 홈에 끼움. 중간 칸막이용 미니픽스 위치를 먼저 확인한 후 조립.

  3. 중간 선반 (라픽스로)

    중간 선반을 라픽스 2 set으로 결합. (2~3단인 경우 반복)

  4. 상판 결합 (미니픽스 + 목다보)

    상판을 미니픽스 4 set + 목다보 4 ea로 결합.

  5. 우라잡이 + 스크류 마감

    뒷판에 우라잡이를 고정하고, 뒷판과 중간 칸막이를 스크류볼트로 고정해 마무리.

3단 책장만 — 다보 + 이동형 선반

📐 다보 위치

3단 책장은 추가로 다보 4ea (전방 뿔다보 2 + 후방 평다보 2)를 조립한 후 이동형 선반을 끼웁니다. 다보 타공 3개 된 부분이 위로 가게 설치, 피스 조립용 구멍이 아래로 가게.

3단 코너 책장만 — 측판 좌·우 + 앞판

코너 책장은 측판이 좌·우 각각 1개씩, 앞판 1ea이 추가됩니다. 조립 순서:

하판 ▶ 측판(좌,우) ▶ 앞판 ▶ 뒷판 ▶ 중간선반 ▶ (반복) ▶ 상판 ▶ 우라잡이
