설치 단계

  1. 프레임 베이스 조립

    프레임 베이스를 미니픽스 set으로 조립합니다.

  2. 풋보드 + 헤드보드 결합

    프레임 베이스에 풋보드와 헤드보드를 미니픽스 + 목다보로 결합합니다.

    ⚠️ 볼트 결합 방향
    • 헤드보드 — 헤드보드 후면에서 볼트 결합
    • 풋보드 — 프레임 베이스 안쪽에서 볼트 결합

    인서트(번데기)는 밖을 향하도록 조립.

    ⛔ 가로다리 구분

    가로다리_헤드측 (관통보링)과 풋보드를 반드시 구분! 헷갈리면 다시 분해해야 합니다.

  3. 상판 결합

    상판 2개를 베이스 프레임에 조립합니다.

    상판 2ea

  4. 쿠션 + 밀림방지 패드 + 코너캡 인계

    쿠션 1ea와 밀림방지 패드를 부착한 후, 풋보드의 코너캡을 고객님께 전달.

    ⚠️ 쿠션 이동 주의

    쿠션을 끌어서 이동하면 안 됩니다. 하부 밀림방지 발통에 의해 오염 및 스크래치 발생. 반드시 들어서 이동해 주세요.

  5. (옵션) 서랍 패널 — 서랍 + 백패널 + 핀 조명

    서랍 내부에서 백 패널과 일반 볼트로 조립. 백 패널에 핀 조명을 M6 볼트로 설치.

  6. (옵션) 수납 패널 — 하판 + 측판 + 상판 + 선반

    하판과 측판 2ea를 미니픽스로 결합. 상판과 내부 선반 2ea를 측판에 미니픽스로 조립.

    측판 2ea 하판 1ea 상판 1ea 내부 선반 2ea

  7. (옵션) 수납 패널 — 백 패널 + 도어

    선반 본품과 백 패널을 미니픽스로 결합. 전면 도어 좌·우 각 1ea를 경첩 3 + 경첩 피스 12로 설치.

    📍 더블 도어 수납장은 경첩 6 + 피스 24 사용.

  8. 전선 + 어댑터 정리

    전선캡 / 바 조명 + 연장선을 정리하고 DC 어댑터에 연결합니다.

    ⚡ 옵션별 전선

    싱글 수납장 · 더블헤드 수납장 모두 전선캡 + 바 조명 + 연장선이 동봉됩니다.

인수 체크리스트

  • 가로다리 방향 (헤드측 관통보링 / 풋보드측)
  • 인서트(번데기) 방향 = 바깥
  • 밀림방지 패드 부착
  • 중앙 스티커 2개 부착
  • 실리콘 코너가드 부착
  • 풋보드 코너캡 → 고객 인계
  • 바 조명 / USB / 터치스위치 동작 확인
