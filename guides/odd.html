설치 단계

조립 순서: 사이드 패널 → 헤드보드 패널 + 전선 → 프레임 하/상 → 풋보드

  1. STEP 1 — 사이드 패널 조립

    사이드 패널(580 또는 480)을 M6×20 평머리 볼트 12ea와 ㄷ브라켓 2ea로 조립합니다.

    M6×20 볼트 12ea ㄷ브라켓 2ea
    📐 사이드 패널 종류
    • 580 서랍형 — 서랍 + M8×35 볼트 2ea로 서랍 결합
    • 580 서랍 + 조명형 — 서랍 + 핀조명(체결너트) + 조명 어댑터
    • 480 선반형 — 선반 + 목다보 2ea + M8×55 하우징 2ea + M8×55 볼트 2ea
    • 480 선반 + 조명형 — 선반 + 핀조명 + 충전단자 어댑터

  2. STEP 2 — 사이드 패널 + 헤드보드 패널 조립 (전선 작업 완료)

    조립한 사이드 패널과 헤드보드 패널을 결합합니다. 이 단계에서 LED 전선 배선을 함께 완료합니다.

    ⚡ 전선 작업 주의

    헤드보드 패널 결합 전에 LED 케이블과 스위치 연결을 완료하세요. 조립 후 배선 수정이 어렵습니다.

    LED 스위치 1ea 어댑터 1ea

  3. STEP 3 — 프레임 하/상 조립

    깔판 하(下)를 프레임에 먼저 결합한 뒤, 깔판 상(上)을 M6×35 + M6×25 볼트로 조립합니다. 다리를 M8×35 볼트로 체결합니다.

    M6×35 볼트 6ea M6×25 볼트 6ea M8×35 볼트 SS:4 / QK:6 / LK:8 ea 밀림방지패드 2ea
    ⚠️ 밀림방지패드

    깔판 상 조립 완료 후 프레임 하단 4모서리에 밀림방지패드 2ea를 부착합니다.

  4. STEP 4 — 풋보드 조립

    풋보드를 M6×20 볼트 4ea + ㄴ브라켓 2ea + 하우징 4ea로 프레임에 결합하여 마무리합니다.

    M6×20 볼트 4ea ㄴ브라켓 2ea 하우징 4ea

인수 체크리스트

  • 사이드 패널 볼트 전체 체결 (M6×20 12ea × 패널 수)
  • LED 전선 연결 및 스위치 동작 확인
  • 다리 M8×35 볼트 끝까지 조임
  • 밀림방지패드 부착
  • 풋보드 ㄴ브라켓 결합 흔들림 없음
  • 서랍 슬라이딩 정상 작동
  • 조명 어댑터 동작 확인
