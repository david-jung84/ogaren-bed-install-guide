룬드빅

수납 파데 + 서랍 파데 + 헤드보드 + 패널 + 조명까지 결합되는 누어 시그니처 모델. 옵션이 많은 만큼 박스별로 분류한 후 시작해 주세요.

📋 8 단계 ⏱ 소요 60~90분 👥 인원 2인 필수 🔧 공구 수동 드라이버
📄 원본 PDF ⚡ 헤드 조명 결선
⛔ 임팩트 드라이버 사용 금지

미니픽스·라픽스·연결볼트 체결 시 임팩트 드라이버 사용 시 부품 파손이 다수 발생합니다. 반드시 수동 드라이버로 조립해 주세요.

목차
  1. 박스 구성 (6개)
  2. 수납 파데 — 마이다·측판
  3. 수납 파데 — 보강목
  4. 서랍 파데 — 지판·측판
  5. 서랍 파데 — 언더레일
  6. 헤드·풋보드 연결
  7. 속서랍 설치
  8. Q확장 파데 (Q/K만)
  9. 패널 + 조명

박스 구성

출고 시 박스가 5~6개 배송됩니다. 시작 전 모두 모여 있는지 확인하세요.

박스내용주요 부품
1 수납형 헤드헤드보드 + 후드 + 어댑터 + 듀얼케이블조명 · USB · 마감스티커 8ea
2 수납 파데지판 ×2 / 측판 ×2 / 중간판 / 마이다 ×2미니픽스 16 set · 목다보 4 / 연결볼트 4 · 너트 2
3 서랍 파데지판(전/후) / 측판 ×4 / 천판 / 뒷판언더레일 3 set · 미니픽스 34 set · 원피스 4
4 서랍속서랍 ×3 / 마에다 ×3시스템 볼트 12 (6×25)
5 패널패널 몸통 + T선반 + 돋보기등연결칩 2 · 원피스 11 · 피스 2
6 Q확장 파데Q/K 사이즈만 포함가로판넬 · 세로판넬 ×2 · 덮개
📦 박스 정렬 팁

설치 순서대로 2 → 3 → 1 → 4 → (6) → 5 박스를 늘어놓으면 동선이 짧아집니다.

설치 단계

  1. 수납 파데 ① 마이다 + 측판 + 중간판

    마이다 1개를 바닥에 놓고, 측판 2개와 중간판 1개를 미니픽스로 조립합니다. 지판 2개를 홈에 끼웁니다.

    ⚠️ 방향 주의

    측판은 미니픽스 하우징이 바깥쪽을 향하도록 두세요. 반대로 두면 다음 단계에서 헤드와 결합 불가.

    미니픽스 set 측판 2ea 중간판 1ea 지판 2ea

  2. 수납 파데 ② 보강목 + 남은 마이다

    보강목 2개를 미니픽스 + 목다보로 조립합니다. 남은 마이다 1개를 추가하면 수납 파데가 완성됩니다. 세워주세요.

    📍 중간 칸막이의 하우징 자리에 마감 스티커 4개를 부착합니다.
    보강목 2ea 목다보 4ea 마감스티커 4ea

  3. 서랍 파데 ① 지판 + 측판 4개

    지판 전면·후면을 바닥에 놓고, 측판 4개를 미니픽스로 결합합니다.

    ⚠️ 헤드 결합 보링 방향

    측판의 헤드 결합용 보링이 지판(전면)을 향하도록 두세요. 전면/후면 구분 필수.

    뒷판(우라)을 후면 지판과 미니픽스로 결합 → 측판 4개와 스크류볼트 6개로 고정.

  4. 서랍 파데 ② 언더레일 + 천판

    언더레일을 측판 유도보링에 맞추어 전용 피스 3개씩 설치합니다. 천판을 미니픽스로 결합.

    📐 슬하 맞추기

    정면 기준으로 좌측·우측 모두 언더레일과 측판이 슬하가 일치해야 서랍이 부드럽게 들어갑니다.

    언더레일 3 set 전용피스 18ea 미니픽스 15 set

  5. SS / Q·K 헤드(풋보드) 결합

    수납 파데를 먼저 설치한 후, 위치를 조정해서 서랍 파데를 설치합니다. 헤드/풋보드를 연결볼트·너트로 체결.

    ⛔ 보링 위치 주의

    수납 파데는 안쪽 보링, 서랍 파데는 바깥쪽 보링에 연결합니다. 반대로 하면 사이즈가 맞지 않습니다.

    파데 ─ 헤드 ─ 후드 ─ 풋보드

  6. 속서랍 설치 (3개)

    속서랍 3개와 수납 파데 덮개 2개를 조립합니다.

    ✓ 속서랍 설치 4단계
    1. 언더레일을 바깥으로 뺀다
    2. 서랍을 레일 위에 올리고 "딸깍" 소리가 날 때까지 당긴다
    3. 한 번 더 "딸깍" 소리가 날 때까지 끝까지 닫는다
    4. 캐처 레버로 수평을 맞춘다
    마에다와 속서랍을 시스템볼트 4개로 결합한 후, 히든선반을 서랍 위에 올립니다.
    🔧 서랍 분리 시

    서랍을 끝까지 연 후 캐처의 방아쇠를 양쪽에서 동시에 당기면 안전하게 분리됩니다.

  7. Q확장 파데 (Q / K 사이즈만)

    가로 판넬과 세로 판넬을 결합하고 덮개를 덮습니다.

    💡 조립이 빡빡하면

    헤드/풋보드 연결볼트가 완전히 조여진 상태에서는 확장 파데 조립이 어렵습니다. 연결볼트를 약간 풀어준 후 조립하면 수월합니다.

  8. 패널 + 조명 설치

    패널에 T선반을 설치 → 뒷판(우라) 분리 → 조명 설치 → 헤드와 패널을 연결칩으로 결합 → 뒷판 재조립.

    T선반 1ea 연결칩 2ea 스크류볼트 3ea 조명 1ea
    ⚡ 조명 결선 주의

    LED 조명과 다른 어댑터를 함께 연결하면 정상 작동하지 않을 수 있습니다. 반드시 동봉된 터치스위치 + USB 포트만 사용하세요. (어댑터 ← 듀얼케이블 ← 터치스위치 + USB 포트 ← LED 바)

최종 체크리스트

인수 전 확인

  • 수납 파데 / 서랍 파데 위치가 안쪽·바깥쪽 보링과 일치
  • 서랍 3개 모두 부드럽게 열고 닫힘 (캐처 "딸깍" 정상)
  • 헤드 조명 ON/OFF 정상 작동
  • USB 충전 포트 정상 (USB-A, USB-C 둘 다)
  • 외부 보링 마감 스티커 8개 부착
  • 밀림방지패드 2개 부착
  • 중간판 마감스티커 4개 부착
