버디 데이베드

유아용 데이베드 4종 (플레인 일반/수납 · 하우스 일반/수납). 모든 파츠와 구조가 동일하며 헤드보드 형상만 다릅니다. 리프트블록과 푹잠가드는 옵션입니다.

설치 단계

조립 순서: (리프트블록) → 깔판 → 가로 다리 → 사이드협 → 헤드·풋 보드

  1. (옵션) 리프트블록 조립

    리프트블록 옵션을 선택한 경우에만 진행합니다. 동봉된 꼬마 드라이버로 미니픽스 8세트로 결합.

    ⚠️ 하우징 방향

    모든 면의 미니픽스 하우징이 내부에 위치하도록 맞춰야 다음 단계에서 깔판과 결합됩니다.

    옵션이 없는 경우 다음 단계로 바로 이동.

  2. 리프트블록 + 깔판 (옵션 선택 시)

    꼬마 드라이버로 리프트블록을 깔판에 미니픽스 16세트로 결합하고, 플라스틱 발통은 스크류로 고정합니다.

    📍 미니픽스 하우징이 노출된 방향에 맞춰 깔판을 뒤집어 결합.

  3. 깔판 + 가로다리

    깔판을 뒤집은 상태에서 가로다리를 미니픽스 8세트로 결합합니다. 리프트블록을 선택하지 않은 경우 이 단계가 시작점.

  4. 사이드협 ① 연결 (벽면 또는 도어)

    사이드협 한 쪽을 바닥에 눕힌 뒤, 앞서 조립한 깔판 파츠를 90° 회전시켜 미니픽스 7세트로 결합합니다.

    📐 결합 요령

    표시된 부분을 잡고 모든 미니픽스 간격에 맞춰 천천히 끼워 결합. 한 번에 누르려 하면 부품이 뜰 수 있습니다.

  5. 사이드협 ② 반대편

    나머지 사이드협도 동일한 방식으로 미니픽스 7세트로 결합합니다.

  6. 헤드보드 + 풋보드

    헤드보드와 풋보드를 미니픽스 12세트로 결합합니다.

    ⚠️ 수납형 헤드 USB 방향

    수납형 헤드의 경우 USB 포트가 정방향에 위치하도록 조정. 위 = USB-A / 아래 = USB-C

  7. 마감 스티커 + 동봉품 인계

    제품을 바로 세운 후 측판 내부에 마감 스티커 7개를 부착하여 완성.

    ✓ 고객 인계품

    동봉된 엽서와 꼬마 드라이버는 향후 분리·재조립 시 필요하므로 고객님께 전달.

(옵션) 푹잠 가드 설치

푹잠 가드 옵션을 선택한 경우 추가로 진행합니다.

  1. 도어 패널 + 브라켓

    도어 패널과 브라켓을 M6 볼트 4개로 결합합니다.

  2. 부직포 스티커 부착

    부직포 스티커 10개를 브라켓 내부 좌·우 측면에 부착합니다.

    📍 부직포 1개는 단독 부착 / 부직포 4개는 2개씩 겹쳐서 볼트 위로 부착

인수 체크리스트

  • 마감 스티커 7개 부착 완료
  • USB-A · USB-C 충전 정상 (수납형 헤드)
  • 흔들림 없이 안정적
  • 고객님께 엽서 + 꼬마 드라이버 전달
  • (옵션) 푹잠 가드 부직포 누락 없는지
